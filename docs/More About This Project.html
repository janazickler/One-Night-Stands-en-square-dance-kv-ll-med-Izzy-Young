More About This Project

This project was carried out as part of the Digitising Cultural Heritage Material module (32LDK1) in the Master's program in Library and Information Science, Digital Library and Information Services BMDD1 12032 HT2021 at Borås University, Sweden. The aim of this project is to create a digitised copy of physical material and make it available online. The digitisation project was carried out by students Ranka Gest and Jana Zickler. The object of digitisation is a printed book written and published in 1983 by Israel Goodman Young with a title One Night Stands – en Square-Dance Kväll med Izzy Young. The aim of the digitization project of this rare manual on traditional American square dances is to preserve this interesting resource and make it accessible to a wider audience. The purpose of this project is to promote the guidance and appreciation of traditional American square dance and to better understand its social and cultural significance. The digitization process involves converting the physical publication into a digital format that can be easily searched, navigated, and downloaded.
